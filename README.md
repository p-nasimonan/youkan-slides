# youkan-slides

LT・技術発表用スライド管理リポジトリ

Marp + Volta + pnpm で構築した、責務に応じて分離・再利用可能なスライド管理システムです。

## 特徴

✨ **柔軟なテンプレート**: 技術紹介・5分LT・自宅サーバーなど、複数のテンプレートから選択
🎨 **カスタムテーマ**: 目に優しいパステル調デザイン（キャラクター同様の柔らかい水色ベース）
🔄 **コンポーネント再利用**: 自己紹介・クロージング・QRコードなどを include で取り込み可能
📁 **スライド分類**: トピック別・イベント別に自動整理
🚀 **自動ビルド・デプロイ**: GitHub Actions で push 時に自動構築・GitHub Pages へ公開
✅ **品質チェック**: PR 時に Markdown リント(面倒だから基本ブランチ分けないと思う)

## セットアップ

### 前提

- Volta（Node.js バージョン管理）
- pnpm（パッケージマネージャー） 

## 使い方

### 1. スライドをプレビュー

```bash
pnpm run dev
```

ブラウザで `http://localhost:8080` を開くと、自動リロード付きでプレビュー可能です。

### 2. 新規スライド作成

```bash
pnpm run new
```

対話式で以下を聞かれます：
- テンプレート選択（tech-intro / lt-5min / base）
- カテゴリ選択（topics/kubernetes など）
- ファイル名・タイトル・発表者名

### 3. スライド一覧表示

```bash
pnpm run list
```

現在のスライドを階層別に一覧表示します。
