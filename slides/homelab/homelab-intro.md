# 自宅サーバー（Homelab）とは？

> 自宅に設置する個人所有のサーバー環境

- 学習・実験用のインフラ環境
- 自分だけのクラウド
- 技術検証の場

# 始め方のヒント

1. **小さく始める**: Raspberry Pi や古いPCから
2. **目的を決める**: 何を学びたいか
3. **段階的に拡張**: 一度に全部やらない
4. **記録を残す**: ブログや構成図
